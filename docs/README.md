# CoC Analytics website (Privacy + landing)

Google Play requires a **public HTTPS URL** for your Privacy Policy. This `docs/` folder is published with **GitHub Pages**.

## Contact

- Email: `piperincon.dev@gmail.com`
- English policy URL (use this in Play Console): `https://imshepo.github.io/coc-analytics/privacy.html`

## GitHub Pages setup

In the repo: **Settings → Pages → Build and deployment**

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

After a minute, open:

- Home: `https://imshepo.github.io/coc-analytics/`
- Privacy: `https://imshepo.github.io/coc-analytics/privacy.html`
- Account deletion (Play Console): `https://imshepo.github.io/coc-analytics/delete-account.html`

## Keep Play Console in sync

If you later add analytics, ads, cloud accounts, or crash reporting, update:

1. This Privacy Policy
2. Play Console → App content → Data safety
3. The “Last updated” date on the policy pages
